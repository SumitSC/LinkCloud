LinkCloud

LinkCloud acts as a cloud storing system, user can create account and upload links which can be further accessed by the user. The links uploaded can be only visible to the user, maintaining user privacy. The user interface gives the user a admin panel for adding, viewing and deleting accounts. User can not only save the links uploaded but also delete particular links that user wants to. User can access the links from anywhere just by loging in with the registered account. LinkCloud is a PHP based system, which gives the user features such as:

  • User friendly UI
  • Account Support
  • Link privacy
  • User only links
  • Deleting links
